This code can be used to make sure a CSV file containing a column name of "food" has all the correct image (.jpg ONLY) files in the stimuli folder. 

It prompts for Mac paths specifically.

It prompts for name and path of the new folder, creates it for you, and moves all images that exist in both the CSV file and the stimuli folder into the new one. 

It can also check for duplicates in the final images folder. 

Enjoy!
